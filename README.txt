# Smart File Renamer 사용법

## 프로그램 소개
Smart File Renamer는 파일 내용을 분석하여 지능적으로 파일명을 변경해주는 프로그램입니다.
PDF, HWP와 같은 문서 파일을 분석하여 과목, 날짜, 문서 유형 등을 추출하고, 사용자가 원하는 패턴으로 이름을 변경할 수 있습니다.

## 실행 방법
1. `SmartFileRenamer.exe` 파일을 실행합니다.
   (초기 실행 시 압축 해제 등으로 인해 시간이 조금 걸릴 수 있습니다.)

## 소스 코드
함께 포함된 소스 코드는 `Source` 폴더에 있습니다.
