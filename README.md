# 📁 스마트 파일 리네이머 (Smart File Renamer)

HWP/PDF 문서를 분석하여 연도, 월, 분류를 자동 추출하고 파일명을 일괄 변경하는 도구입니다.

![Smart File Renamer](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Python](https://img.shields.io/badge/python-3.9+-green.svg)
![License](https://img.shields.io/badge/license-MIT-yellow.svg)

## ✨ 주요 기능

- **📄 문서 분석**: HWP, HWPX, PDF 파일에서 텍스트 추출
- **🔍 자동 분류**: 연도, 월, 과목/분류 자동 인식
- **✨ 스마트 추출**: 기존 키워드에 없어도 문서 상단에서 자동 감지
- **📝 블록 패턴**: 드래그 앤 드롭으로 파일명 패턴 편집
- **✏️ 수동 편집**: 추출 정보 직접 수정 가능
- **📁 다른 폴더로 복사**: 원본 유지하고 다른 위치로 복사
- **↩️ 되돌리기**: 마지막 작업 취소 가능
- **🏷️ 사용자 키워드**: 자주 사용하는 분류 저장

## 📥 설치 및 실행

### 방법 1: 실행 파일 (권장)

1. [Releases](../../releases) 에서 최신 버전 다운로드
2. `SmartFileRenamer.zip` 압축 해제
3. `SmartFileRenamer.exe` 실행

### 방법 2: Python 소스 실행

```bash
# 저장소 클론
git clone https://github.com/YOUR_USERNAME/SmartFileRenamer.git
cd SmartFileRenamer

# 의존성 설치
pip install -r requirements.txt

# 실행
python main.py
```

## 📋 사용 방법

### 1. 파일 선택

- **드래그 앤 드롭**: 파일/폴더를 창에 끌어다 놓기
- **폴더 버튼**: 폴더 전체 선택
- **파일 버튼**: 개별 파일 선택

### 2. 패턴 설정

블록 버튼을 클릭하여 파일명 패턴 구성:

- `{Year}` - 연도 (예: 2024)
- `{Month}` - 월 (예: 03)
- `{Subject}` - 분류 (예: 물리)
- `{Original}` - 원본 파일명
- `{Seq}` - 순번 (01, 02, ...)

### 3. 미리보기 확인

- 자동으로 새 파일명 미리보기 표시
- 추출정보가 틀리면 더블클릭하여 수정

### 4. 실행

- **✅ 실행** 버튼 클릭
- 저장 위치 지정 시 해당 폴더로 복사
- 미지정 시 원본 위치에서 이름만 변경

## 🔒 지원 파일 형식

| 확장자 | 설명 |
|--------|------|
| `.hwp` | 한글 97-2020 문서 |
| `.hwpx` | 한글 2022+ 문서 (OOXML) |
| `.pdf` | PDF 문서 |

## ⚙️ 기술 스택

- **GUI**: PySide6 (Qt for Python)
- **PDF 분석**: pdfminer.six
- **HWP 분석**: olefile + 자체 파서
